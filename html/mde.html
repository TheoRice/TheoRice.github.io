MDE
MDE is an open-source, cross-platform MP4 metadata editor (metadata editor, MDE, get it?). Similar to programs like Subler or Wondershare's Video Convertor but available across all platforms and completely open-source! I am building it with ElectronJS, Bootstrap, and utilizing AtomicParsley It is currently still in development, build 0.5.0 Banana, but should be in working condition if you want to check out the github repo. Read more about the development on my blog!
